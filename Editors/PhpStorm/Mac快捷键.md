## 2. 编辑

`cmd + /` 单行注释

`command + option + /` 多行注释

`cmd + x` 删除当前行

`cmd + d` 复制当前行

`cmd + shift + u` 切换大小写



## 3. 跳转

`cmd + I` 跳到某行

`cmd + e` 列出打开文件

`cmd + b` 跳到声明处



## 4. 搜索

`cmd + 1` 列出左侧文件 

`cmd + 6` 查找//@todo标签

`cmd + 7`显示类中的方法

`cmd + shift + F`  全局查找

`cmd + 鼠标点击`  函数追踪





## 5. 格式与备注

`cmd + alt + L`  格式化代码

`cmd + alt + shift + L`  格式化文件代码
